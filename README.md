# concompte

personal project for having quotations and invoice

## prerequisite

- [node js >= 5](https://nodejs.org/en/)
- [couchdb](https://couchdb.apache.org/)

## build

### first and always step 

```
npm install
```

### dev server

```
npm run dev
```
